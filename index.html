# Obsidian Helm Repository This is a Helm chart repository for deploying Obsidian on Kubernetes. ## Usage ```bash helm repo add obsidian https://thinking-and-coding.github.io/obsidian-helm-chart helm repo update helm install my-obsidian obsidian/obsidian ``` ## Documentation For detailed documentation, visit the [main repository](https://github.com/thinking-and-coding/obsidian-helm-chart). ## Charts This repository is automatically updated by GitHub Actions when new releases are tagged.
